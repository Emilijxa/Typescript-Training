What is TypeScript

TypeScript is like a puzzle helper! Imagine you're putting together a puzzle, and...

1. Setting Up TypeScript in Your Project

Before you write TypeScript, you need to install it. You use a tool called npm or yarn to do this. Just open your terminal and type: npm install -g typescript.

Change .js files to .ts:

tsc fileName.ts

2. Types Are Important!

In JavaScript, you don’t have to say what type of data (numbers, words, true/false) something is. But with TypeScript, you tell it exactly what something is, and it keeps you from making mistakes. 

let age: number = 10;

3. TypeScript Helps You With Objects!

Example: Here’s how you can describe a person:


interface Person {
  name: string;
  age: number;
}

let friend: Person = { name: "Laura", age: 12 };

4. Classes and Objects

TypeScript adds “classes” that describe objects in detail, like blueprints for houses:


class Car {
  make: string;
  year: number;

  constructor(make: string, year: number) {
    this.make = make;
    this.year = year;
  }

  drive() {
    console.log("Engine sound");
  }
}

let myCar = new Car("BMW", 2020);
myCar.drive();

5. Superpowers (Generics and Unions)

Generics: It's like making a magic box that can hold any type of thing, but still checks it’s correct.

function magicBox(item: T): T { return item; }

Unions: Allow more than one type of thing:

let id: string | number;  // Can be a number OR a word
